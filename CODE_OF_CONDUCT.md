# Code of Conduct

We want CodeMate to be a practical, welcoming open-source project.

Participants are expected to:

- Be respectful and constructive.
- Assume good intent while still being precise about technical concerns.
- Avoid harassment, insults, and personal attacks.
- Keep discussions focused on the project and its users.

Project maintainers may remove comments, issues, or pull requests that violate these expectations.
